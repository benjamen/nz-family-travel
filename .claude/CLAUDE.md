# NZ Family Travel

SEO content site for NZ family travel — activities, destinations, itineraries. Python static site generator.

## What it does
Travel guide targeting families visiting or living in NZ. Content clusters around regions, activities, school holidays, and road trips.

## Structure
- `build.py` — SSG build script
- `content/` — markdown content pages
- `data/` — structured data (destinations, activities)
- `layouts/` — Jinja2 templates
- `SEO-ACTION-PLAN.md` — current SEO priorities
